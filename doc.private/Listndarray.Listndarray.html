add_one lst1 lst2 is the sum of all elements of lst1 added up with lst2 Requires:

  • lst1 and lst2 should be 1-D lists
  • shape lst1 = shape lst2

Examples:

  • add [1;2;3] [0;0;0] is 1;2;3
  • add [] [] is 0

Raises:

  • failure if shape lst1 <> shape lst2
